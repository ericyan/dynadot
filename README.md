# Dynadot

A Ruby interface to the [Dynadot's Domain API](https://www.dynadot.com/domain/api2.html).

## Installation

Add this line to your application's Gemfile:

    gem 'dynadot'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dynadot
